Every component in every state, on the Thready theme (base #B6E376, accessible accent #446E12 light / #B6E376 dark, teal secondary #ABDDC9/#B7EBD6). Toggle the theme (top right) to audit light and dark — the toggle stamps data-theme exactly as the shared ThemeService does. Sections marked VERIFIED reproduce shipped vasic-digital/design_system CSS verbatim; sections marked PROPOSED are [DEFAULT — adjustable] upstream candidates. See platform-map.md and README.md.
