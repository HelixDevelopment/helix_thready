⬇ Drop a channel link

Dropping text/URL matching t.me/… or max.ru/join/… opens the Add-Channel wizard at step 3 with the link pre-filled (grounded: wizard step 3, wireframes §3.4).
Dropping media files has no ground-truth ingest path — posts and assets originate from messenger channels — so file ingest is [OPEN: THREADY-DES-16], not designed here.

Custom title bar per OS chrome only — web UI unchanged

  • macOS: traffic lights leading; global system menu bar; title bar is a data-tauri-drag-region; fullscreen behavior native.
  • Windows: caption buttons ─ ▢ ✕ trailing; in-window menu row; snap layouts on ▢ hover (native).
  • Linux: client-side decorations mirroring Windows layout; respects the desktop environment’s dark preference.
  • Window title = Thready — <active Account> (account switcher, wireframes §3.1).

Tray behavior [OPEN: THREADY-DES-08] · [DEFAULT — adjustable]

  • Close (✕) hides to tray; polling/WS subscriptions keep running so processing continues to stream.
  • Tray icon badges the count of active processing jobs; failure turns the badge danger.
