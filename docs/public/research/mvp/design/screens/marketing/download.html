Terminal (TUI / CLI)

pattern verified

A Go client on Bubble Tea + Lipgloss, styled from the same design tokens — the pattern is already proven in-house. Web and CLI are the first-ship surfaces. The install commands below are the proposed channel; release binaries are not published yet.

# proposed channel [DEFAULT — adjustable] — binaries not yet published

# or: download a release binary, then
